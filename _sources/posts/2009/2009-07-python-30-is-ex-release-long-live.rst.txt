Python 3.0 is an ex-release! Long live Python 3.1!
==================================================

*This was originally posted on blogger* `here <https://pyfound.blogspot.com/2009/07/python-30-is-ex-release-long-live.html>`_.

`There will be no Python 3.0.2 release or further 3.0.x maintenance/security
releases. <http://mail.python.org/pipermail/python-list/2009-July/718561.html>`_
All users of Python 3.0 are strongly encouraged to upgrade to `Python
3.1 <http://www.python.org/download/releases/3.1/>`_.
