Benjamin Peterson Receives PSF Community Service Award
======================================================

*This was originally posted on blogger* `here <https://pyfound.blogspot.com/2011/09/benjamin-peterson-receives-psf.html>`_.

  
The Python Software Foundation has presented a `Community Service
Award <http://www.python.org/community/awards/psf-awards/>`_ for the first
quarter of 2011 to `Benjamin Peterson <http://www.benjamin-peterson.org/>`_ for
his work as Release Manager and the numerous contributions to the Python Core.  
The Python Software Foundation is pleased to recognize Benjamin's
contributions to the community.
