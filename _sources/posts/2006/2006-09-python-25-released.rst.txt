Python 2.5 released
===================

*This was originally posted on blogger* `here <https://pyfound.blogspot.com/2006/09/python-25-released.html>`_.

Today release manager Anthony Baxter announced:

> On behalf of the Python development team and the Python community, I'm happy
> to announce the FINAL release of Python 2.5.

The new release can be downloaded from `the Python 2.5
page <http://www.python.org/2.5/>`_. A description of the new features in Python
2.5 can be found in the article `"What's New in Python
2.5" <http://docs.python.org/whatsnew/whatsnew25.html>`_, or you can read `the
release notes <http://www.python.org/download/releases/2.5/NEWS.txt>`_ for every
last detail.

A note for journalists: if you want to talk to a PSF director about the new
release, please send us an e-mail at and a director will contact you to
arrange a discussion.
